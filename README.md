# chuchu-scrapper

## Description
`chuchu-scrapper` is a web scraping tool designed to extract data from various websites efficiently. It automates the process of collecting information, making it easier to gather and analyze large datasets from the web.

## Features
- Extracts data from multiple websites with customizable scraping rules.
- Supports pagination and dynamic content scraping.
- Handles rate-limiting and retries for robust data collection.
- Outputs data in various formats (JSON, CSV, etc.).

## Requirements
- Python 3.8 or higher
- Poetry for dependency management

## Installation
1. Clone the repository:
'''bash
git clone https://github.com/Egbewattnkongho13/chuchu-scrapper.git '''
